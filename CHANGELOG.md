# Changelog

## 0.3.0-beta - 2026-03-06

- Added Montreal-first location-aware optimization.
- Added FastAPI backend for web/mobile integration.
- Added SQLite persistence for users and saved plans.
- Added token-based auth with login/refresh/logout/logout-all.
- Added token expiry/revocation and cleanup endpoint.
- Added schema-versioned migrations for safer DB evolution.
- Added saved-plan CRUD flows: create, paginated list, single fetch, rename, delete.
- Added dynamic-location web prototype with auth and saved-plan management.
- Added CI workflow, Docker/Compose scaffolding, and deployment docs.
